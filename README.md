# mybmi.ai

Free BMI, BMR, TDEE and Body Fat calculators with AI-powered health plans.

## Stack
- React 19 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Recharts, jsPDF
- PWA (offline-capable)

## Dev
npm install && npm run dev

## Build
npm run build

## Data
All data stored locally in localStorage. No backend, no account required.
